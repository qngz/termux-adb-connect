Интерактивный bash-скрипт для быстрого подключения по ADB к Android-устройствам 
(Android 11+) через беспроводную отладку.

## Требования
- Termux (скрипт рассчитан на запуск именно в нём — на других окружениях 
  поведение непредсказуемо)
- Android 11+ на подключаемом устройстве
- Включённый режим разработчика и "Отладка по Wi-Fi" на целевом устройстве
- Оба устройства должны быть в одной сети (или используйте 127.0.0.1 
  для подключения к тому же устройству, на котором запущен скрипт)

## Использование
```bash
bash adb-connect.sh
```
